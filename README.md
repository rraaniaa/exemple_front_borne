# Ordermate Pro

Kiosk de commande en libre‑service.

## Démarrage rapide

Prérequis : Node.js + npm.

```sh
npm install
npm run dev
```

## Scripts utiles

- `npm run dev` : serveur de développement
- `npm run build` : build de production
- `npm run preview` : prévisualisation du build

## Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
